1. How-To Guide — Install Visual Studio Code (Windows)

Audience: New developers and students on Windows

Summary: Step-by-step installation instructions with verification steps and troubleshooting tips.

  1. Visit the official Visual Studio Code website.
  2. Click Download for Windows and run the downloaded .exe.
  3. Follow the installer: accept agreement → choose install folder → enable Add to PATH → Install.
  4. Launch VS Code and confirm installation via Help → About.

2. User Guide — To‑Do List Web App

Audience: Non-technical end users

Summary: Concise feature overview, step-by-step tasks, and a troubleshooting section.

How to add a task

  1. Click Add Task.
  2. Enter the task name and click Save.

Troubleshooting

  • Tasks not saving — ensure input is not empty and try refreshing the page.
  • App not loading — check your internet connection or try a different browser.
